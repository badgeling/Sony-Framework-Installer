### Background:
Made by Badgeling, Sony Framework Installer is Sony's framework files for any Android device. Allowing you to install and update Sony apps from Play Store. 
Please check[GitHub thread here.](https://github.com/badgeling/Sony-Framework-Installer)

### Installation:

1. Download and flash the Sony Framework Installer through TWRP or Magisk Manager.
2. Reboot.
3. Clear Play Store's data.
4. Open Play Store and install your preferred Sony apps.
